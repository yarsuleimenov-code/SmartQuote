Fuel Prices

Назначение: хранить fuel prices, используемые pricing engine.

Current Avg. является внешним значением. Fuel Surcharge % задаётся админом.

Цель страницы

Variables.html используется для управления pricing engine: коэффициентами, fuel prices, vehicle logic и operational rules.

Брокер не должен взаимодействовать с variables напрямую.

Основной принцип

  • Variables изменяются только admin role.
  • Estimate хранит variables snapshot.
  • Старые estimates не пересчитываются после изменения variables.
  • Pricing engine использует только active formula version.

Что важно для разработки

  • Variables должны храниться централизованно.
  • Нужен formula versioning.
  • Нужен estimate-level variables snapshot.
  • Fuel prices должны обновляться через API.
  • Vehicle selection должен выполняться backend pricing engine.
