Purpose

My Estimates is used to review generated customer-facing estimates. These are not working drafts.

Draft vs Estimate

  • Draft: editable working quote.
  • Estimate: customer-facing version that should be versioned.
  • Changes after sending should create a new estimate version.

Main Actions

  • Preview HTML/PDF: open the frozen customer estimate snapshot.
  • Breakdown: open internal cost explanation for admin review.
  • Reopen Draft: create an editable copy for changes.
  • Status actions: track sent, approved, rejected, and negotiation state.

Future Logic

  • PDF / link generation.
  • Email / SMS sending.
  • Estimate expiration rules.
  • Client view tracking.
  • Invoice, Order, and eBOL workflow after shared storage and handoff rules are defined.
  • Estimate to Invoice through Stripe / Square.
  • Version history for all post-send changes.
